<Section icon={Map} title="Fundamentos da Build de Velocidade no Build a Hooper">

Uma build de velocidade no Build a Hooper deve parecer explosiva desde a primeira posse de bola. O objetivo não é apenas correr sem rumo; é obter separação rápida, movimentação limpa e controle suficiente para transformar velocidade em ataque real. Na prática, as melhores builds de velocidade geralmente se encaixam na categoria de armadores (guards) ou alas (wings), onde o controle de bola, passes e explosão podem moldar cada posse de bola.

É aqui que a lógica do criador de jogadores do jogo faz a diferença. Você tem um limite de pontos de atributos, então a velocidade deve apoiar um propósito claro em vez de ser uma estatística isolada. Se a sua build se move rapidamente, mas não consegue finalizar, arremessar ou sobreviver à pressão, a vantagem desaparece rápido. As builds de velocidade mais fortes mantêm uma via principal de pontuação e uma habilidade de apoio.

<Callout type="tip" title="O que a Velocidade Realmente Deve Fazer">
A velocidade é mais valiosa quando cria separação, abre ângulos de passe e ajuda você a se recuperar na defesa. Trate-a como uma ferramenta, não como a build inteira.
</Callout>

<Section icon={Target} title="Melhores Prioridades de Atributos para Jogadores Rápidos">

A combinação certa de atributos depende de você querer um criador puro, um armador pontuador ou um ala bidirecional. Para a maioria dos jogadores, a velocidade deve liderar, mas não deve ser a única estatística importante. O sistema de atributos do Build a Hooper recompensa o equilíbrio dentro de uma função focada, especialmente quando você quer um jogador que ainda tenha desempenho sob pressão.

Uma regra útil é simples: cada ponto em velocidade deve ter uma função. Se ajudar você a passar pelos defensores, ótimo. Se também melhorar a defesa de transição ou a recuperação rápida, melhor ainda. Mas assim que sua movimentação for forte o suficiente, os próximos pontos devem apoiar seu ataque principal.

| Atributo | Prioridade | Por Que Importa | Não Negligencie |
|---|---|---|---|
| Velocidade | Muito Alta | Melhora a separação, transição e recuperação | Explosão sem controle é fácil de defender |
| Controle de Bola | Muito Alta | Permite converter velocidade em infiltrações limpas | Velocidade sozinha não supera a pressão |
| Arremesso | Alta | Pune defensores que recuam demais na infiltração | Infiltradores unidimensionais ficam sem espaço |
| Finalização | Alta | Transforma caminhos abertos em pontos | Velocidade vazia parece boa, mas pontua menos |
| Passes | Média-Alta | Ajuda armadores a sobreviverem a dobras e armadilhas | Bons criadores precisam de uma rota de escape |
| Defesa | Média-Alta | Mantém jogadores rápidos úteis sem a bola | Ataque puro pode virar alvo fácil |
| Perfil Físico | Média | Apoia o tamanho, alcance e encaixe de confrontos | Escolhas corporais ruins podem limitar a função |

<Callout type="info" title="Regra de Melhor Prioridade">
Jogadores rápidos geralmente têm melhor desempenho quando a velocidade, o controle de bola e uma habilidade de pontuação sobem juntos. Uma estatística de elite é mais fraca do que três úteis.
</Callout>

A build de velocidade mais segura para a maioria dos jogadores é aquela que ainda funciona quando a primeira opção é anulada. Isso significa ter pelo menos uma resposta de apoio: um arremesso após o drible (pull-up), um passe ou um corte forte em direção ao aro. Uma build rápida sem recursos de apoio pode render ótimos momentos em destaques, mas ainda parecer travada no fluxo real do jogo.

<Section icon={Sparkles} title="Como Configurar a Build Passo a Passo">

Se você quer uma build de velocidade refinada, siga uma configuração estruturada em vez de gastar pontos aleatoriamente. O draft e o fluxo da carreira recompensam os jogadores que conhecem sua identidade desde cedo. Assim que sua função estiver clara, o restante da build se tornará mais fácil de ajustar.

<Steps>
  <Step num="1" title="Escolha a função principal">
    Decida se seu jogador irá criar, pontuar ou defender primeiro. Uma build de velocidade precisa de um propósito claro, especialmente nas posições de armador e ala.
  </Step>
  <Step num="2" title="Defina o núcleo de movimentação">
    Coloque a velocidade e o controle de bola no topo das suas primeiras prioridades. Isso lhe dá uma base jogável antes de gastar em atributos secundários.
  </Step>
  <Step num="3" title="Adicione uma arma de pontuação">
    Escolha o arremesso ou a finalização como sua principal forma de punir os defensores. Não deixe a build apenas com movimentação e nenhuma ameaça de cesta.
  </Step>
  <Step num="4" title="Corrija os pontos fracos">
    Use os pontos restantes para cobrir a defesa, os passes ou a estabilidade física. Isso mantém a build útil quando os oponentes forçarem você a jogadas desconfortáveis.
  </Step>
</Steps>

| Ponto de Decisão | Se Sim | Se Não |
|---|---|---|
| Você quer controlar a bola frequentemente? | Priorize o controle estilo armador principal | Mude para uma build de ala |
| Você quer pontuar em espaço aberto? | Invista em velocidade e arremesso | Foque mais em finalização |
| Você quer um valor de equipe mais seguro? | Adicione defesa e passes | Mantenha a build mais agressiva |
| Você quer um estilo de jogo mais simples? | Escolha um armador infiltrador | Escolha um ala bidirecional |

<Callout type="success" title="Regra Simples de Configuração">
Uma boa build de velocidade deve responder a três perguntas rapidamente: como ela cria espaço, como ela pontua e como ela sobrevive à pressão.
</Callout>

| Passo | Objetivo | O Que Você Deve Notar |
|---|---|---|
| Identidade da build | Escolher uma função | O jogador já deve parecer especializado |
| Núcleo de movimentação | Adicionar velocidade e controle | Você deve vencer as corridas de pé logo cedo |
| Arma principal | Adicionar uma via de ataque | Os defensores devem respeitar sua escolha |
| Camada de proteção | Adicionar defesa ou passes | A build ainda deve funcionar sob pressão |

O principal erro aqui é focar demais em lances de efeito. Se você buscar apenas a explosão e ignorar o resto, a build pode parecer ótima em quadra aberta, mas desmoronar assim que a defesa se ajustar. A configuração mais forte é aquela que ainda consegue pontuar depois que a primeira opção de infiltração é cortada.

</Section>

<Section icon={Shield} title="Erros Comuns Que Deixam a Build Lenta">

A maioria das builds de velocidade fracas falha pelos mesmos motivos: muitas prioridades divididas, falta de pontuação de apoio e pouca resistência à pressão. O jogo lhe dá espaço suficiente para criar um especialista útil, mas não o bastante para desperdiçar pontos em tudo. Isso significa que a disciplina importa.

Use esta lista de verificação antes de salvar sua build. Se vários itens parecerem incertos, geralmente é um sinal de que a função do jogador ainda está ampla demais.

<Callout type="warning" title="O Que Geralmente Prejudica Mais a Build">
Uma build de velocidade fica mais fraca quando cada ponto é gasto buscando uma identidade diferente. A especialização geralmente é mais forte do que a variedade.
</Callout>

Outro problema comum é ignorar como a build será realmente usada durante uma campanha de carreira. Se o seu jogador foi feito para guiar a posse de bola, os passes importam mais. Se o seu jogador foi feito para atacar espaços abertos, a finalização ou o arremesso precisam de suporte. Uma build de velocidade inteligente não é apenas rápida; ela tem um propósito.
