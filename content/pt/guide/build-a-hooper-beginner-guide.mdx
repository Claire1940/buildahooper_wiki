<Section icon={Map} title="Guia para Iniciantes Build a Hooper: O Básico da Primeira Build">

Build a Hooper foca na criação de um jogador de basquete personalizado, na escolha de uma posição e na moldagem de uma carreira do zero. Para iniciantes, a jogada mais inteligente não é buscar todas as estatísticas. É focar em uma função clara e construir em torno dela. O sistema estilo NBA do jogo funciona melhor quando seu primeiro jogador tem um propósito.

<Callout type="tip" title="Comece com uma função, não com uma lista de desejos">
Uma build de iniciante deve ser focada. Se você quer pontuar, construa para pontuar. Se você quer rebotes e controle do garrafão, foque em tamanho e jogo interior.
</Callout>

<Section icon={Target} title="Melhores Escolhas e Configuração da Primeira Build">

Depois de conhecer sua função, o próximo passo é escolher uma build que corresponda ao seu nível de conforto. Build a Hooper suporta múltiplas identidades de jogador, mas iniciantes devem evitar distribuir pontos de forma muito rasa. Uma build inicial forte geralmente tem uma força principal e uma força secundária.

<Callout type="warning" title="Evite o superinvestimento precoce">
Não jogue pontos em todas as categorias. Uma build dispersa parece flexível no papel, mas muitas vezes parece mais fraca na jogabilidade real.
</Callout>

  <Step num="3" title="Gaste pontos por prioridade">

  <Step num="4" title="Teste a build no modo carreira">

<Section icon={Shield} title="Posições e Prioridades de Atributos">

As posições moldam como você deve pensar sobre cada ponto gasto. As quatro funções principais de Build a Hooper são fáceis de entender, mas cada uma muda a forma como você deve abordar o ataque, a defesa e as estatísticas físicas. Iniciantes se saem melhor quando a posição e o plano de atributos estão alinhados desde o início.

<Callout type="success" title="Combine os atributos com a função">
Um armador não deve parecer um pivô, e um pivô não deve ser construído como um arremessador minúsculo. Mantenha a função e as estatísticas alinhadas.
</Callout>

| Posição | Função Típica | Melhor Foco de Atributo |
| --- | --- | --- |
| Armador (PG) | Principal facilitador | Passe, velocidade, arremesso |
| Ala-Armador (SG) | Pontuador de perímetro | Arremesso, movimentação, pontuação |
| Ala (SF) | Ala versátil | Ataque e defesa equilibrados |
| Pivô (C) | Âncora de garrafão | Tamanho, rebotes, defesa |

| Grupo de Atributos | O Que Suporta | Prioridade do Iniciante |
| --- | --- | --- |
| Ataque | Pontuação e criação | Alta se você quer pontuar |
| Defesa | Paradas e contestações | Alta se quer uma build de duas vias |
| Físico | Tamanho e perfil atlético | Muito importante para a identidade da função |
| Build equilibrada | Flexibilidade em quadra | Melhor para novos jogadores |

Uma regra simples funciona bem: se estiver na dúvida, construa o tipo de jogador que você gosta de assistir no basquete real. Isso mantém sua primeira carreira mais agradável e torna suas escolhas de build mais fáceis de lembrar.

| Erro Comum | Melhor Abordagem |
| --- | --- |

| Ignorar o perfil físico | Use o tamanho para reforçar a build |
| Gastar uniformemente em tudo | Priorize as duas principais estatísticas primeiro |
| Copiar jogadores avançados | Construa para sua própria curva de aprendizado |
