<Section icon={Target} title="Build a Hooper アスレチックビルドのアイデンティティ">

Build a Hooper のアスレチックビルドは、素早い動き、リムへのプレッシャー、そして積極的なディフェンスを、毎ポゼッションで信頼できる価値に変えることがすべてです。最高のビルドは単に速いだけでなく、接触に耐えうる強さと、ボールを持っていない時でも貢献できる十分なディフェンス力を兼ね備えています。

このビルドを考える上で最も簡単な方法は、ステータスではなく「役割」を優先することです。リムを攻めるのか、複数のポジションを守るのか、あるいはペリメーターからチャンスを作るのかを決め、その役割に合わせて体格と能力値を形成しましょう。

<Section icon={Sparkles} title="実際に成果が出る能力値の優先順位">

アスレチックビルドでは、投資する順番を絞ることが報われます。序盤の価値は通常、プレーが崩れる前にディフェンダーを抜き去るのに役立つ特性から生まれます。つまり、スピード、加速、フィニッシュ、そして穴にならない程度のディフェンス力です。

最も安全なアプローチは、段階的にポイントを消費することです。まず簡単なポゼッションを生み出すものから始め、次に数回のアップグレード後もビルドの有効性を維持できるものを追加します。優れたアスレチックビルドは、単に数値が強いだけでなく、動いている時に常に使い心地が良いものであるべきです。

| 能力値グループ | 優先度 | 重要な理由 |
|---|---|---|
| スピード | 高 | セパレーションとトランジションのチャンスを作る |
| 加速 | 高 | ビルドが素早くトップスピードに乗るのを助ける |
| フィニッシュ | 高 | 身体能力の優位性を得点に変える |
| ディフェンス | 中〜高 | 毎ポゼッションでビルドの有用性を維持する |
| パス | 中 | ファストブレイクとシンプルな読みをサポートする |
| シュート | 中 | 下がりすぎたディフェンダーを罰する |

| 消費プラン | 最適な用途 | 選択すべき時 |
|---|---|---|
| 60/25/15 | スピード、フィニッシュ、サポート能力 | シンプルな得点を求める新規プレイヤーに最適 |
| 50/30/20 | バランスの取れたアスレチックプレー | 2ウェイウィングに最適 |
| 40/35/25 | クリエイターとアスリートのミックス | ボールを保持する機会が多い場合に最適 |

<Callout type="warning" title="1つのステータスに偏りすぎない">
純粋なスコアリング特化は序盤は強く見えますが、ビルドが単調になりがちです。ゲームのスピードが上がった時でも対応できるよう、十分なディフェンスとパス能力を確保しておきましょう。
</Callout>

</Section>

<Section icon={Shield} title="サイズ、ウィングスパン、ポジションのトレードオフ">

体格の設定は、アスレチックビルドが接触状況でどのように振る舞うかを決定します。体が小さいほどドライブ時のスピードとキレが増し、体が大きいほど接触を吸収し、トラフィックを抜けてフィニッシュし、より多くのシュートをコンテストできます。鍵となるのは、自分がどう勝ちたいかに合ったボディタイプを選ぶことです。

最も安全なルートを選びたいなら、ウィングサイズのフレームが通常最高のバランスを提供します。純粋な瞬発力が欲しいなら小さく、ペイント内に住みたいならスピードの目標値が維持できる範囲でサイズを大きくしましょう。

| ボディタイプ | メリット | デメリット | 最適なフィット |
|---|---|---|---|
| スモールガード | 圧倒的な瞬発力、容易なセパレーション | リーチが短い、接触フィニッシュが弱い | メインのボールハンドラー |
| ウィングフレーム | バランスの取れたスピードと長さ | 特定の分野で突出していない | ほとんどのアスレチックビルド |
| コンパクトビッグ | 強力なリバウンド、優れたインサイドプレー | ペリメーターでのリカバリーが遅い | リムランナーまたはペイントディフェンダー |

| ポジション選択 | アスレチックな強み | 共通の弱点 |
|---|---|---|
| ポイントガード | 最高のオープンコートスピード | スイッチ時に狙われやすい |
| シューティングガード | 強力なスコアリングレーン | 十分なプレイメイクサポートが必要 |
| スモールフォワード | 最高のオールラウンドなプロファイル | 規律ある能力値の使用が求められる |
| センター | インサイドのパワーとリバウンド | ペリメーターの機動力が低い |

<Callout type="tip" title="推奨されるデフォルトの選択">
ほとんどのプレイヤーにとって、ウィングサイズのアスレチックビルドは、プレイスタイルを固定しすぎることなく、スピード、フィニッシュ、ディフェンスのリーチの最適な組み合わせを提供してくれます。
</Callout>

</Section>

<Section icon={Map} title="アスレチックビルド作成のステップバイステップ">

クリーンなセットアップはポイントの無駄を防ぎ、最初のセッションからビルドを強く感じさせてくれます。次に何をアップグレードすべきか常に把握できるよう、以下の順序に従ってください。

<Steps>
  <Step num="1" title="役割を選ぶ">
    リムプレッシャー、2ウェイウィングディフェンス、または機動力のあるインサイドプレーの中から、主な仕事を1つ選びます。最初からすべてのスタイルをカバーしようとしないでください。
  </Step>
  <Step num="2" title="まず体格を決める">
    役割をサポートするフレームを決定します。瞬発力ならスモール、バランスならウィング、ペイントでの存在感が必要な場合のみ大きくします。
  </Step>
  <Step num="3" title="コア能力値を構築する">
    贅沢なステータスよりも先に、スピード、加速、フィニッシュを向上させます。コートに立ち続けられるよう、早い段階でディフェンスも追加しましょう。
  </Step>
  <Step num="4" title="テストと洗練">
    数試合プレイしてみて、弱点を補強します。リムまで到達できないなら瞬発力を、失点が多いならディフェンスのサポートを追加します。
  </Step>
</Steps>

| よくある間違い | 悪影響 | 修正方法 |
|---|---|---|
| 早すぎる段階でのシュート重視 | アスレチックな個性を損なう | スピードとフィニッシュを最優先する |
| ディフェンスの無視 | ターゲットにされやすくなる | コンテスト、スティール、またはブロックのサポートを追加 |
| プレイヤーのサイズを大きくしすぎる | 瞬発力とリカバリー能力が低下 | 機動力がしっくりくるまで体を絞る |
| ポイントを均等に分散させる | 平均的なビルドになってしまう | 1つの核となる勝利条件にコミットする |

<Callout type="info" title="セットアップのルール">
今すぐポゼッションを勝ち取るのに役立たないステータスは、後回しにしましょう。アスレチックビルドは、最初のアップグレードがディフェンスに即座にプレッシャーを与えられるようになった時に真の強さを発揮します。
</Callout>

</Section>
