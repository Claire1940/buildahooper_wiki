正しい身体パラメータを選択することは、**build a hooper スモールフォワードビルド** の基礎となります。身長の1インチ、体重の1ポンドが最大能力値の上限に直接影響します。以下の構成により、高いスピードを維持しながらエリートアニメーションを解除できます。

| 体重 | 215 lbs | スピードを犠牲にすることなく、フィジカルな守備に必要な強さを提供する |
| ウィングスパン | 7'0" | シュートの安定性と、ブロックやスティールのリーチのバランスを取る |

身体フレームが確立されたら、特定のバッジの閾値に達するように能力値を割り当てる必要があります。これらの閾値をわずか1ポイントでも下回ると、重要なゴールドや殿堂入り（Hall of Fame）バッジがロックされてしまいます。

<Section icon={Target} title="ステップバイステップ：build a hooper スモールフォワードビルドの割り当て">

キャラクター作成プロセス中に能力ポイントを効率的に割り当てるために、以下の構造化された手順に従ってください。この順序により、重要度の低いカテゴリーにポイントを浪費することを防げます。

<Steps>
  <Step num="1" title="身体属性の設定">
    身長 6'8"、体重 215 lbs、ウィングスパン 7'0" を設定します。身体の動きは後からアップグレードする際に最もコストがかかるため、まずスピードと加速の上限を最大化します。
  </Step>
  <Step num="2" title="シュートとプレイメイキングの最大化">
    ミドルレンジを 88、3ポイントを 85 まで上げます。次に、ボールハンドルを 77、スピード（ボール保持時）を 75 まで上げます。これにより、初日からスペースを作り出し、安定してシュートを決めることができます。
  </Step>
  <Step num="3" title="ディフェンスとフィニッシュの配分">
    外郭守備が 85 に達するまでポイントを割り当て、次にスティールを 73 まで上げます。ブロックを 78 まで上げてディフェンスセクションを完了します。フィニッシュについては、信頼性の高いスラッシュアニメーションを確保するためにドライビングダンクを 85 に設定します。
  </Step>
  <Step num="4" title="キャップブレイカーの適用">
    ゲームプレイのマイルストーンを通じてキャップブレイカーを獲得したら、5ポイントすべてをスティールレーティングに適用します。これによりスティールが 73 から 78 に上昇し、より高いティアのディフェンスバッジが解除されます。
  </Step>
</Steps>

<Callout type="info" title="キャップブレイカー戦略">
ディフェンスバッジは初期作成時に到達するのが難しい高い能力投資を必要とするため、キャップブレイカーをディフェンス統計に使用するのは非常に効率的です。
</Callout>

</Section>

<Section icon={Sparkles} title="必須バッジとアニメーション">

バッジは、実際の試合シナリオでプレイヤーがどのようにパフォーマンスを発揮するかを決定します。能力値が高くても、正しいバッジ設定がなければ苦戦することになります。パフォーマンスを最大化するために、以下の特定のバッジのレベル上げに集中してください。

| バッジ名 | 推奨ティア | 発動条件 |
| :--- | :--- | :--- |
| エージェント3 | シルバー | 3ポイントラインからのプルアップジャンプシュート |
| ミディ・マジシャン | ゴールド | ミドルレンジからのフェイダウェイやプルアップ |
| クランプ | シルバー | L2/LTボタンを押し込んだ激しいディフェンス姿勢の維持 |
| インターセプター | シルバー | パスレーン上でのパスのスティール |
| ポスタライザー | シルバー | ペイント内でのコンタクトダンクの試行 |

アニメーションパッケージも同様に重要です。間違ったドリブルスタイルやジャンプシュートのリリースは、プレイヤーの動きを鈍く感じさせます。スピードを最適化するために、以下の推奨アニメーションを使用してください。

| アニメーションタイプ | 推奨スタイル | 必要能力値 |
| :--- | :--- | :--- |
| ジャンプシュート | カスタムベース | 3ポイント 85+ |
| ドリブルスタイル | プロ | スピード（ボール保持時） 75+ |
| サイズアップ | ノーマル | ボールハンドル 75+ |
| エスケープパッケージ | ベーシック | ボールハンドル 70+ |
| レイアップパッケージ | ロングアスリート | ドライビングレイアップ 80+ |

<Callout type="success" title="バッジのシナジー">
「ミディ・マジシャン」バッジと「スペースクリエイター」バッジを組み合わせてください。この組み合わせにより、ステップバックからのミドルレンジジャンプシュートは、ディフェンダーが効果的にコンテスト（妨害）するのがほぼ不可能になります。
</Callout>
