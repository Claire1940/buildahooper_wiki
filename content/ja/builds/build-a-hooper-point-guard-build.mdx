<Section icon={Target} title="スコアリングPGの属性優先順位">

ここで多くのプレイヤーがスターを作るか、ポイントを無駄にするかが決まります。Build a Hooperにおいて、ポイントガードは早い段階で影響の低い分野に多額の投資をすべきではありません。最初の目標は、毎回のポゼッションを楽にすることです。クリーンなドリブル、安全なパス、そしてディフェンダーを正直にさせるのに十分なシュート力です。

<Callout type="warning" title="この間違いを避ける">
開始時にポイントを均等に分散させないでください。分散したガードはどこも平均的で、どこもエリートではなくなり、序盤の進行が大幅に遅くなります。
</Callout>

| 属性グループ | 優先度 | 理由 |
|---|---|---|
| ボールハンドリング | **高** | よりクリーンな仕掛けとコントロールを可能にする |
| パス | **高** | チームメイトを関与させ、ターンオーバーを減らす |
| シュート | **高** | プルアップの脅威によりディフェンダーに警戒させる |
| ディフェンス | 中 | ペリメーターで生き残るために必要 |
| フィジカル | 中 | スピード、スタミナ、コンタクトの安定性を支える |

| ビルドタイプ | 第一フォーカス | 第二フォーカス |
|---|---|---|
| オフェンス型PG | **ハンドリング + シュート** | パス |
| バランス型PG | **ハンドリング + パス** | ディフェンス |
| ディフェンス型PG | **ディフェンス + スピード** | 基本的なショットクリエイション |
| 初心者向けPG | **パス + シュート** | フィジカル |

<Callout type="info" title="実践的なルール">
2つのスタッツが同等に感じられる場合は、より多くのポゼッションで役立つ方を選んでください。コートに立つたびに向上を感じられるポイントガードビルドが、通常はより良い投資となります。
</Callout>

<Section icon={Sparkles} title="作成手順とドラフト設定">

最もクリーンなビルドパスはシンプルです。役割を定義し、物理プロファイルを設定し、ゲームプランをサポートする数値を洗練させます。Build a Hooperのキャリアの流れは、ビルドを確定させる前に自分が何をしたいかを知っているプレイヤーに報います。これにより、ドラフト設定と序盤の進行が非常にスムーズになります。

<Steps>
  <Step num="1" title="ガードのアイデンティティを選択する">
    パサー、スコアラー、または攻守のオーガナイザーのどれを作るか決めます。この役割が後のすべての決定を導くはずです。
  </Step>
  <Step num="2" title="物理プロファイルを設定する">
    自分のスタイルをサポートする体型を選びます。速いガードは通常、クリーンな動きを求め、頑丈なガードはコンタクトと安定性を重視します。
  </Step>
  <Step num="3" title="コア属性を固定する">
    最初の主要なポイントをハンドリング、パス、シュートに投入します。これらのスタッツが、ガードがどれだけ有用なポゼッションを生み出せるかを左右します。
  </Step>
  <Step num="4" title="ビルドのロジックをテストする">
    そのビルドがオフェンスを開始し、プレッシャーを打破し、ミスを罰することができるか自問してください。答えが「ノー」なら、進む前に再調整しましょう。
  </Step>
</Steps>

| 設定項目 | 推奨される方向性 | 結果 |

| シュートパス | **極端ではなく信頼性重視** | 優れたスペーシング |
| ディフェンスパス | **ペリメーター重視** | 安全なマッチアップ |
| フィジカルパス | **スピードとスタミナ** | ポゼッション価値の向上 |

ドラフト準備が整ったガードとは、すぐにチームを助けることができる選手です。シュートを作り、ボールを動かし、プレッシャー下でも使い物になるビルドであれば、正しい軌道に乗っています。

<Section icon={Shield} title="バッジ、進行、および週次目標">

バッジと進行は、すでに選択した役割を増幅させるものであるべきです。ポイントガードはすべてのアップグレードパスを一度に必要とはしません。ボールの安全性、効率的な得点、そしてクリーンなプレイメイキングをサポートするものが必要です。焦点を絞ることで、ビルドはより早く強力に感じられるようになります。

<Callout type="tip" title="アップグレード順序">
メインのアクションを改善するバッジカテゴリーから始めましょう。毎回のポゼッションでハンドリング、パス、シュートを行うなら、それらが最優先されるべきシステムです。
</Callout>

| バッジカテゴリー | 優先度 | 最善の用途 |
|---|---|---|
| プレイメイキング | **高** | より速いクリエイションとクリーンなアシスト |
| シュート | **高** | より信頼性の高いシュートセレクション |
| ディフェンス | 中 | ガードとしての封じ込め力の向上 |
| インサイドフィニッシュ | 低〜中 | ドライブを多用する役割の場合のみ有用 |
| ユーティリティ | 中 | 弱点を補うのに役立つ |
