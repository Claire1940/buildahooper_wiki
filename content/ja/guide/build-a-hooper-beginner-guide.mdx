<Section icon={Target} title="おすすめの最初のビルド選択とセットアップ">

自分の役割が決まったら、次は自分の好みに合ったビルドを選びましょう。Build a Hooperでは多様なプレイスタイルがサポートされていますが、初心者はポイントを分散させすぎないように注意すべきです。優れた最初のビルドは、通常「1つのメインの強み」と「1つのサブの強み」を持っています。

<Callout type="warning" title="序盤の投資分散を避ける">
すべてのカテゴリーにポイントを注ぎ込まないでください。器用貧乏なビルドは、数値上は万能に見えても、実際のゲームプレイでは器用貧乏で弱く感じられがちです。
</Callout>

</Steps>

<Section icon={Shield} title="ポジションと能力値の優先順位">

選択するポジションによって、ポイントの割り振りの考え方が変わります。Build a Hooperの4つの主要な役割はシンプルですが、それぞれオフェンス、ディフェンス、フィジカルステータスへのアプローチが異なります。初心者は、最初からポジションと能力値の計画を一致させておくとうまくいきます。

<Callout type="success" title="能力値を役割に合わせる">
ガードがセンターのように動く必要はありませんし、センターを小さなシューターのように作るべきでもありません。役割とステータスの方向性を一致させましょう。
</Callout>

| ポジション | 代表的な役割 | 最も重視すべき能力値 |
| --- | --- | --- |
| ポイントガード | プライマリ・ハンドラー | パス、スピード、シュート |
| シューティングガード | ペリメーター・スコラー | シュート、ムーブメント、スコアリング |
| スモールフォワード | 万能型ウィング | バランスの取れたオフェンス＆ディフェンス |
| センター | インサイド・アンカー | サイズ、リバウンド、ディフェンス |

| 能力値グループ | サポートするプレイ | 初心者の優先度 |
| --- | --- | --- |
| オフェンス | 得点とシュートクリエイト | 得点したいなら「高」 |
| ディフェンス | ストップとコンテスト | 攻守万能（ツーウェイ）を目指すなら「高」 |
| フィジカル | サイズと身体能力 | 役割のアイデンティティにおいて「極めて重要」 |
| バランス型ビルド | コート全体の柔軟性 | 新規プレイヤーに「最適」 |

シンプルなルールとして「迷ったら、実際のバスケ観戦で自分が好きなタイプの選手を真似て作る」のがおすすめです。これにより、最初のキャリアがより楽しくなり、ビルドの選択に迷わなくなります。

| よくある間違い | 推奨されるアプローチ |
| --- | --- |
| 多くの役割を混ぜすぎる | 1つの役割を決め、それを強化する |
| フィジカル特性を無視する | サイズを活かしてビルドを補強する |
| すべてに平均的にポイントを振る | まず上位2つのステータスを最優先する |
| 上級者のビルドをそのまま真似る | 自分の上達スピードに合わせたビルドを作る |

</Section>

<Section icon={Sparkles} title="キャリア、ドラフト、そして序盤の進行">

Build a Hooperのキャリアモードこそが、あなたの最初の決断が試される場所です。このゲームは、選手を作成し、スキルを磨き、長期的なバスケのアイデンティティを確立していく進行システムを採用しています。そのため、最初の数時間の選択が、その後のプレイをどれだけ快適にするかを左右します。

<Callout type="info" title="長期的な視点を持つ">
初心者の最善の戦略は、単に「強くスタートする」ことだけではありません。自分が理解し、改善し、長く楽しめる選手を作ることです。
</Callout>

<Steps>
  <Step num="1" title="選手を作成する">
    ポジションを選択し、ビルドの方向性を決め、ガード、ウィング、ビッグマンのどれにするかを決定します。
  </Step>
  <Step num="2" title="キャリアパスに入る">
    NBAスタイルの進行ループに入り、プレイを通じて経験値を獲得し始めます。
  </Step>
  <Step num="3" title="強みをさらに伸ばす">
    ゼロから作り直すのではなく、自分の役割にすでに適しているスキルをさらに強化します。
  </Step>
  <Step num="4" title="次のバージョンを洗練させる">
    数回プレイした後、何が強みで何が弱みだったかをメモし、次のビルド作成に役立てます。
  </Step>
</Steps>

| 進行ステージ | 目標 | 学べること |
| --- | --- | --- |
| 作成 | 役割の決定 | ビルドシステムの仕組み |
| 序盤の試合 | リズムを掴む | どのステータスが最も重要か |
| 中盤の進行 | 強みを研ぎ澄ます | 特化させる方法 |
| 長期キャリア | レガシーを築く | 選手が長期的にどう成長するか |
